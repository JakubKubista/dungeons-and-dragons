Detailed Description

Trida reprezentujici Zbroj tridy Hrdina.

Trida je pevne spata se tridou Hrdina. Trida prebira hodnoty do atributu ze tridy InventarZbroj. Hrdina tuto Zbroj vlastni pevne jiz od zacatku hry. Prepisovani hodnot v atributech ma vypadat jako nahrazovani jednotlivych Zbroji.
