Detailed Description

Abstraktni Trida reprezentujici jednotlive Nepratele (NPC) po mape v Mistnostech.

Zde je videt typicky polymorfismus, kde abstraktni trida NPC dedi svoje ciste virtualni metody do svych potomku, ktere tedy jeste prebiraji jeji atributy.
